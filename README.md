# エラーコード検索アプリ

製造機器（インバーター、サーボアンプ等）のエラーコードを入力すると、エラー内容と処置方法を即座に表示するローカル完結Webアプリです。

## 特徴

- **完全オフライン** — 外部通信なし、127.0.0.1 のみで動作
- **誰でも使える .exe 配布** — Python/Node 不要、ダブルクリックで起動
- **複数CSVに対応** — メーカー/カテゴリ/製品ごとのCSVを自動検出
- **列構成に依存しない** — 3列でも4列でも正しく表示
- **即時検索** — コード入力のたびに瞬時フィルタ

## データフォルダ構成

```
<データルート>/
├── 三菱/
│   ├── インバーター/
│   │   └── A800.csv
│   └── サーボアンプ/
│       └── J4.csv
└── <他メーカー>/
    └── <カテゴリ>/
        └── <製品名>.csv
```

CSVの第1列を「エラーコード」として扱います。残り列はヘッダ名をそのまま表示します。

## 使い方（.exe 配布版）

1. `error-code-viewer.exe` をダブルクリック
2. 初回はフォルダ選択画面が開く → CSVが入っているルートフォルダを指定
3. ブラウザが自動で開く
4. メーカー → カテゴリ → 製品を選び、エラーコードを入力

## 開発環境セットアップ（自PC）

```bat
setup.bat
```

Python 3.10以上 と Node.js 18以上 が必要です。

## 起動（開発/自PC）

```bat
start.bat
```

`http://127.0.0.1:8000` をブラウザで開きます。

## .exe のビルド

```bat
build_exe.bat
```

`dist/error-code-viewer.exe` が生成されます（Python/Node 不要な配布用単体 exe）。

## テスト

```bat
REM バックエンド
cd backend && ..\.venv\Scripts\pytest tests\ -v

REM フロントエンド
cd frontend && npm test
```
